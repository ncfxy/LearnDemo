# 微服务

## 常用微服务框架

- Spring Boot<https://spring.io/projects/spring-boot/>
  - 最流行的JVM框架
- Micronaut<https://micronaut.io/>
  - 基于JVM
  - 主打启动快，低内存，适用serverless应用
- Quarkus<https://quarkus.io/>
  - 为GraalVM和HotSpot量身定制的Kubernetes Native Java框架
- helidon<https://helidon.io/#/>
  - Oracle开发
  - 开发微服务应用的libraries集合
