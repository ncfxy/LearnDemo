# Linux

## Linux 常用命令

| 命令    | 作用                                 |
| ------- | ------------------------------------ |
| sed     | 利用脚本来处理文本文件               |
| which   | 定位可执行文件的路径                 |
| whereis | 查找二进制（命令）、源文件、man 文件 |
| ps -aux | 查看进程                             |
| top     | 类似任务管理器                       |
| free    | 内存使用情况查看                     |
