# Git 

## Git 常用命令
    
    git clone xxxxx
    git branch branchName
    git branch -d branchName
    git commit -m "message"
    git commit --amend          修改最后一个commit
    git add filePath/fileName
    git status

## 一些技巧
    type null > .gitkeep     生成.gitkeep文件
